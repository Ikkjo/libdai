\doxysubsection{Detailed Description}
Calculates both types of \doxylink{classdai_1_1BP}{BP} messages and their normalizers from an \doxylink{classdai_1_1InfAlg}{Inf\+Alg}. 

\doxylink{classdai_1_1BP__dual}{BP\+\_\+dual} calculates "{}dual"{} versions of \doxylink{classdai_1_1BP}{BP} messages (both messages from factors to variables and messages from variables to factors), and normalizers, given an \doxylink{classdai_1_1InfAlg}{Inf\+Alg}. These are computed from the variable and factor beliefs of the \doxylink{classdai_1_1InfAlg}{Inf\+Alg}. This class is used primarily by \doxylink{classdai_1_1BBP}{BBP}.
