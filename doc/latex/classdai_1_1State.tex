\doxysubsection{Detailed Description}
Makes it easy to iterate over all possible joint states of variables within a \doxylink{classdai_1_1VarSet}{Var\+Set}. 

A joint state of several variables can be represented in two different ways, by a map that maps each variable to its own state, or by an integer that gives the index of the joint state in the canonical enumeration.

Both representations are useful, and the main functionality provided by the \doxylink{classdai_1_1State}{State} class is to simplify iterating over the various joint states of a \doxylink{classdai_1_1VarSet}{Var\+Set} and to provide access to the current state in both representations.

As an example, consider the following code snippet which iterates over all joint states of variables {\itshape x0} and {\itshape x1\+:} 
\begin{DoxyCode}{0}
\DoxyCodeLine{VarSet\ vars(\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{x0}},\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{x1}}\ );}
\DoxyCodeLine{\textcolor{keywordflow}{for}(\ \mbox{\hyperlink{classdai_1_1State_aab03f6a5dc8d23f79e0966698f783178}{State}}\ S(vars);\ S.valid();\ S++\ )\ \{}
\DoxyCodeLine{\ \ \ \ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{cout}}\ <<\ \textcolor{stringliteral}{"{}Linear\ state:\ "{}}\ <<\ S.get()\ <<\ \textcolor{stringliteral}{"{},\ x0\ =\ "{}}\ <<\ S(\mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{x0}})\ <<\ \textcolor{stringliteral}{"{},\ x1\ =\ "{}}\ <<\ S(\mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{x1}})\ <<\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{endl}};}
\DoxyCodeLine{\}}

\end{DoxyCode}


\begin{DoxyNote}{Note}
The same functionality could be achieved by simply iterating over the linear state and using \doxylink{namespacedai_afcfc874f85b4a2944ec128ebd1dd19f1}{dai\+::calc\+State()}, but the \doxylink{classdai_1_1State}{State} class offers a more efficient implementation.

A \doxylink{classdai_1_1State}{State} is very similar to a \doxylink{classdai_1_1multifor}{dai\+::multifor}, but tailored for \doxylink{classdai_1_1Var}{Var} \textquotesingle{}s and \doxylink{classdai_1_1VarSet}{Var\+Set} \textquotesingle{}s.
\end{DoxyNote}
