\doxysubsection{Detailed Description}
A \doxylink{classdai_1_1ClusterGraph}{Cluster\+Graph} is a hypergraph with variables as nodes, and "{}clusters"{} (sets of variables) as hyperedges. 

It is implemented as a bipartite graph with variable (\doxylink{classdai_1_1Var}{Var}) nodes and cluster (\doxylink{classdai_1_1VarSet}{Var\+Set}) nodes. One may think of a \doxylink{classdai_1_1ClusterGraph}{Cluster\+Graph} as a \doxylink{classdai_1_1FactorGraph}{Factor\+Graph} without the actual factor values. \begin{DoxyRefDesc}{Todo}
\item[\mbox{\hyperlink{todo__todo000008}{Todo}}]Remove the \+\_\+vars and \+\_\+clusters variables and use only the graph and a contextual factor graph. \end{DoxyRefDesc}
