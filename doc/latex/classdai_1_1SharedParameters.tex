\doxysubsection{Detailed Description}
Represents a single factor or set of factors whose parameters should be estimated. 

To ensure that parameters can be shared between different factors during EM learning, each factor\textquotesingle{}s values are reordered to match a desired variable ordering. The ordering of the variables in a factor may therefore differ from the canonical ordering used in lib\+DAI. The \doxylink{classdai_1_1SharedParameters}{Shared\+Parameters} class combines one or more factors (together with the specified orderings of the variables) with a \doxylink{classdai_1_1ParameterEstimation}{Parameter\+Estimation} object, taking care of the necessary permutations of the factor entries / parameters.
