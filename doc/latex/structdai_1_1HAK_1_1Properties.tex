Enumeration of possible cluster choices. 

The following cluster choices are defined\+:
\begin{DoxyItemize}
\item MIN minimal clusters, i.\+e., one outer region for each maximal factor
\item DELTA one outer region for each variable and its Markov blanket
\item LOOP one cluster for each loop of length at most {\itshape \doxylink{structdai_1_1HAK_1_1Properties_aeaecf3d77c6e6a2036cc652056783523}{Properties\+::loopdepth}}, and in addition one cluster for each maximal factor
\item BETHE Bethe approximation (one outer region for each maximal factor, inner regions are single variables)
