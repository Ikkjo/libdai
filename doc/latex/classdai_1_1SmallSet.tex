class dai\+::\+Small\+Set$<$ T $>$}
Represents a set; the implementation is optimized for a small number of elements. 

\doxylink{classdai_1_1SmallSet}{Small\+Set} uses an ordered {\ttfamily std\+::vector\texorpdfstring{$<$}{<}}{\itshape T}{\ttfamily \texorpdfstring{$>$}{>}} to represent a set; this is faster than using a {\ttfamily std\+::set\texorpdfstring{$<$}{<}}{\itshape T}{\ttfamily \texorpdfstring{$>$}{>}} if the number of elements is small.
