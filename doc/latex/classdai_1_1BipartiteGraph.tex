\doxysubsection{Detailed Description}
Represents the neighborhood structure of nodes in an undirected, bipartite graph. 

A bipartite graph has two types of nodes\+: type 1 and type 2. Edges can occur only between nodes of different type. Nodes are indexed by an unsigned integer. If there are \doxylink{classdai_1_1BipartiteGraph_a6b4f272b57126b4dfb572674eba5c035}{nr\+Nodes1()} nodes of type 1 and \doxylink{classdai_1_1BipartiteGraph_a709376c998449338193e10f9c2250e25}{nr\+Nodes2()} nodes of type 2, the nodes of type 1 are numbered 0,1,2,...,\doxylink{classdai_1_1BipartiteGraph_a6b4f272b57126b4dfb572674eba5c035}{nr\+Nodes1()}-\/1 and the nodes of type 2 are numbered 0,1,2,...,\doxylink{classdai_1_1BipartiteGraph_a709376c998449338193e10f9c2250e25}{nr\+Nodes2()}-\/1. An edge between node {\itshape n1} of type 1 and node {\itshape n2} of type 2 is represented by a Edge({\itshape n1},{\itshape n2}).

A \doxylink{classdai_1_1BipartiteGraph}{Bipartite\+Graph} is implemented as a sparse adjacency list, i.\+e., it stores for each node a list of its neighboring nodes. More precisely\+: it stores for each node of type 1 a vector of \doxylink{structdai_1_1Neighbor}{Neighbor} structures (accessible by the \doxylink{classdai_1_1BipartiteGraph_a0b31bd0db6aff70fe8ba245882783607}{nb1()} method) describing the neighboring nodes of type 2; similarly, for each node of type 2 it stores a vector of \doxylink{structdai_1_1Neighbor}{Neighbor} structures (accessibly by the \doxylink{classdai_1_1BipartiteGraph_a40516848225be41f68b6381d04f8db25}{nb2()} method) describing the neighboring nodes of type 1. Thus, each node has an associated variable of type Bipartite\+Graph\+::\+Neighbors, which is a vector of \doxylink{structdai_1_1Neighbor}{Neighbor} structures, describing its neighboring nodes of the other type. \begin{DoxyRefDesc}{Idea}
