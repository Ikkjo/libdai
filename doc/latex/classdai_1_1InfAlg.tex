\doxysubsection{Detailed Description}
\doxylink{classdai_1_1InfAlg}{Inf\+Alg} is an abstract base class, defining the common interface of all inference algorithms in lib\+DAI. 

\begin{DoxyRefDesc}{Idea}
\item[\mbox{\hyperlink{ideas__ideas000004}{Idea}}]General marginalization functions like \doxylink{namespacedai_aab8aabfe0fcfd4b4bd24757c101449d5}{calc\+Marginal()} now copy a complete \doxylink{classdai_1_1InfAlg}{Inf\+Alg} object. Instead, it would make more sense that they construct a new object without copying the \doxylink{classdai_1_1FactorGraph}{Factor\+Graph} or \doxylink{classdai_1_1RegionGraph}{Region\+Graph}. Or they can simply be made methods of the general \doxylink{classdai_1_1InfAlg}{Inf\+Alg} class. 

Use a \doxylink{classdai_1_1PropertySet}{Property\+Set} as output of an \doxylink{classdai_1_1InfAlg}{Inf\+Alg}, instead of functions like \doxylink{classdai_1_1InfAlg_a915a659034eb4c85abde912d684f8dee}{max\+Diff()} and \doxylink{classdai_1_1InfAlg_a89986bbc1a42554905c6f417f2227ac5}{Iterations()}. \end{DoxyRefDesc}
