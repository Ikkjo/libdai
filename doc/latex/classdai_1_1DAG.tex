\doxysubsection{Detailed Description}
Represents the neighborhood structure of nodes in a directed cyclic graph. 

A directed cyclic graph has nodes connected by directed edges, such that there is no directed cycle of edges n1-\/\texorpdfstring{$>$}{>}n2-\/\texorpdfstring{$>$}{>}n3-\/\texorpdfstring{$>$}{>}...-\/\texorpdfstring{$>$}{>}n1. Nodes are indexed by an unsigned integer. If there are \doxylink{classdai_1_1DAG_a612bc2529939865352ae7d2aa4ec3566}{nr\+Nodes()} nodes, they are numbered 0,1,2,...,\doxylink{classdai_1_1DAG_a612bc2529939865352ae7d2aa4ec3566}{nr\+Nodes()}-\/1. An edge from node {\itshape n1} to node {\itshape n2} is represented by a Edge({\itshape n1},{\itshape n2}).

\doxylink{classdai_1_1DAG}{DAG} is implemented as a sparse adjacency list, i.\+e., it stores for each node a list of its parents and a list of its children. Both lists are implemented as a vector of \doxylink{structdai_1_1Neighbor}{Neighbor} structures (accessible by the \doxylink{classdai_1_1DAG_ada7483211efef87be9d2f9749cb3f3ea}{pa()} and \doxylink{classdai_1_1DAG_afc96dd9895801ea2e092493a1a18bb70}{ch()} methods). Thus, each node has two associated variables of type DAG\+::\+Neighbors, which are vectors of \doxylink{structdai_1_1Neighbor}{Neighbor} structures, describing their parent and children nodes.
