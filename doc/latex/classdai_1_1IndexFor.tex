\doxysubsection{Detailed Description}
Tool for looping over the states of several variables. 

The class \doxylink{classdai_1_1IndexFor}{Index\+For} is an important tool for indexing Factor entries. Its usage can best be explained by an example. Assume {\itshape index\+Vars}, {\itshape for\+Vars} are both \doxylink{classdai_1_1VarSet}{Var\+Set} \textquotesingle{}s. Then the following code\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classdai_1_1IndexFor_a7ce4ecff14943ca0e4303d309e1e79d3}{IndexFor}}\ i(\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{indexVars}},\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{forVars}}\ );}
\DoxyCodeLine{\textcolor{keywordtype}{size\_t}\ iter\ =\ 0;}
\DoxyCodeLine{\textcolor{keywordflow}{for}(\ ;\ i.valid();\ i++,\ iter++\ )\ \{}
\DoxyCodeLine{\ \ \ \ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{cout}}\ <<\ \textcolor{stringliteral}{"{}State\ of\ forVars:\ "{}}\ <<\ \mbox{\hyperlink{namespacedai_afcfc874f85b4a2944ec128ebd1dd19f1}{calcState}}(\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{forVars}},\ iter\ )\ <<\ \textcolor{stringliteral}{"{};\ "{}};}
\DoxyCodeLine{\ \ \ \ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{cout}}\ <<\ \textcolor{stringliteral}{"{}state\ of\ indexVars:\ "{}}\ <<\ \mbox{\hyperlink{namespacedai_afcfc874f85b4a2944ec128ebd1dd19f1}{calcState}}(\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{indexVars}},\ \textcolor{keywordtype}{size\_t}(i)\ )\ <<\ \mbox{\hyperlink{namespacedai_a94b4dd96324af3c0511584a0925f9c15}{endl}};}
\DoxyCodeLine{\}}

\end{DoxyCode}
 loops over all joint states of the variables in {\itshape for\+Vars}, and {\ttfamily (size\+\_\+t)i} equals the linear index of the corresponding state of {\itshape index\+Vars}, where the variables in {\itshape index\+Vars} that are not in {\itshape for\+Vars} assume their zero\textquotesingle{}th value. \begin{DoxyRefDesc}{Idea}
\item[\mbox{\hyperlink{ideas__ideas000012}{Idea}}]Optimize all indices as follows\+: keep a cache of all (or only relatively small) indices that have been computed (use a hash). Then, instead of computing on the fly, use the precomputed ones. Here the labels of the variables don\textquotesingle{}t matter, but the ranges of the variables do. \end{DoxyRefDesc}
