\doxysubsection{Detailed Description}
Stores the data structures needed to efficiently update the approximation of an off-\/tree factor. 

The \doxylink{classdai_1_1TreeEP}{Tree\+EP} object stores a \doxylink{classdai_1_1TreeEP_1_1TreeEPSubTree}{Tree\+EPSub\+Tree} object for each off-\/tree factor. It stores the approximation of that off-\/tree factor, which is represented as a distribution on a subtree of the main tree.
