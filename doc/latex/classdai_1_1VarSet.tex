\doxysubsection{Detailed Description}
Represents a set of variables. 

\begin{DoxyNote}{Note}
A \doxylink{classdai_1_1VarSet}{Var\+Set} is implemented using a \doxylink{classdai_1_1SmallSet}{Small\+Set$<$\+Var$>$} instead of the more natural std\+::set$<$\+Var$>$ because of efficiency reasons. That is, internally, the variables in the set are sorted ascendingly according to their labels. 
\end{DoxyNote}
