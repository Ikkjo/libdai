\doxysubsection{Detailed Description}
Represents the neighborhood structure of nodes in an undirected graph. 

A graph has nodes connected by edges. Nodes are indexed by an unsigned integer. If there are \doxylink{classdai_1_1GraphAL_a7a3919f7d8fb2292cd84eb46c1c716c2}{nr\+Nodes()} nodes, they are numbered 0,1,2,...,\doxylink{classdai_1_1GraphAL_a7a3919f7d8fb2292cd84eb46c1c716c2}{nr\+Nodes()}-\/1. An edge between node {\itshape n1} and node {\itshape n2} is represented by a Edge({\itshape n1},{\itshape n2}).

\doxylink{classdai_1_1GraphAL}{Graph\+AL} is implemented as a sparse adjacency list, i.\+e., it stores for each node a list of its neighboring nodes. The list of neighboring nodes is implemented as a vector of \doxylink{structdai_1_1Neighbor}{Neighbor} structures (accessible by the \doxylink{classdai_1_1GraphAL_a439308c5d55cb48afecbcf70a3623899}{nb()} method). Thus, each node has an associated variable of type Graph\+AL\+::\+Neighbors, which is a vector of \doxylink{structdai_1_1Neighbor}{Neighbor} structures, describing its neighboring nodes.
