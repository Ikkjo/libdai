\doxysubsection{Detailed Description}
Error handling in lib\+DAI is done by throwing an instance of the \doxylink{classdai_1_1Exception}{Exception} class. 

The \doxylink{classdai_1_1Exception}{Exception} class inherits from std\+::runtime\+\_\+error. It defines several types of exceptions and corresponding error messages. The recommended way to throw an instance of the \doxylink{classdai_1_1Exception}{Exception} class is by using the DAI\+\_\+\+THROW or \doxylink{exceptions_8h_a29c0e906e26c7484c268d111c71f79da}{DAI\+\_\+\+THROWE} macros. \begin{Desc}
