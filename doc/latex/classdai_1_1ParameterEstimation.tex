\doxysubsection{Detailed Description}
Base class for parameter estimation methods. 

This class defines the general interface of parameter estimation methods.

Implementations of this interface (see e.\+g. \doxylink{classdai_1_1CondProbEstimation}{Cond\+Prob\+Estimation}) should register a factory function (virtual constructor) via the static \doxylink{classdai_1_1ParameterEstimation_a3248cc18f8683a95e7d318e5c7d27258}{register\+Method()} function. This factory function should return a pointer to a newly constructed object, whose type is a subclass of \doxylink{classdai_1_1ParameterEstimation}{Parameter\+Estimation}, and gets as input a \doxylink{classdai_1_1PropertySet}{Property\+Set} of parameters. After a subclass has been registered, instances of it can be constructed using the \doxylink{classdai_1_1ParameterEstimation_a38ee418f4d5ac9136f13a087528a8718}{construct()} method.

Implementations are responsible for collecting data from a probability vector passed to it from a \doxylink{classdai_1_1SharedParameters}{Shared\+Parameters} container object.

The default registry only contains \doxylink{classdai_1_1CondProbEstimation}{Cond\+Prob\+Estimation}, named "{}\+Cond\+Prob\+Estimation"{}.
